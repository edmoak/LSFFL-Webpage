# LSFFL Website Design Guide

## Project Information

**Project:** Lamad Squad Fantasy Football League Website  
**Platform:** MyFantasyLeague (MFL) + GitHub Pages  
**Repository:** LSFFL-Webpage

---

# Project Goals

Create a modern sports website while continuing to use MyFantasyLeague as the fantasy football engine.

GitHub Pages is the master source for:

- CSS
- JavaScript
- Images
- Documentation
- Custom pages
- Future enhancements

# Coding Standards

- Keep code organized.
- Separate CSS by purpose.
- Keep JavaScript modular.
- Comment complex code.
- Test changes on GitHub before MFL.

---

# Development Workflow

1. Build on GitHub.
2. Test on GitHub Pages.
3. Move finished work to MFL.
4. Keep GitHub as the master copy.
