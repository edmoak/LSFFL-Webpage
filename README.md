# Lamad Squad Fantasy Football League Website

Welcome to the official LSFFL Website project.

This repository contains all custom assets, styling, documentation, and future enhancements for the Lamad Squad Fantasy Football League website.

---

## Project Overview

The LSFFL website is built using two platforms:

- **MyFantasyLeague (MFL)** – League management, scoring, standings, drafts, transactions, and gameplay.
- **GitHub Pages** – Custom website assets including CSS, JavaScript, images, documentation, and future custom pages.

GitHub serves as the master source for all custom development.
